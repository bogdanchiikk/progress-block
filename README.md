# Progress Block Component

Прототип блока Progress для мобильных web-приложений.

## Функциональность

- Отображение прогресса в виде круговой шкалы
- Управление состоянием через API
- Адаптивный дизайн под ориентацию экрана
- Состояния: Normal, Animated, Hidden

## Технологии

- Pure JavaScript
- CSS3
- HTML5
